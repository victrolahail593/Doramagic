# exp01 问题集模板

**版本**：v1  
**用途**：用于记录 `exp01` 的标准问题集，支撑 A/B 对比  
**建议题量**：10–15 题

---

## 使用原则

- 问题必须覆盖不同知识类型
- 问题表述应保持简洁、明确、可比较
- 不要让实验组获得问题文本之外的额外提示
- 如果题目依赖仓库背景，应显式写出前提

## 分类建议

1. 领域概念题
2. 工作流题
3. 规则 / 条件题
4. 风险 / 踩坑题
5. 方案 / 迁移建议题

## 题目设计检查清单

- 是否覆盖 WHAT / HOW / IF 至少三类知识
- 是否至少包含 2 个容易诱发幻觉的问题
- 是否至少包含 2 个需要结构化理解的问题
- 是否避免纯主观问题
- 是否足以区分“泛化回答”和“项目定制回答”
