## 发布内容（4 个修复）

1. **setup_packages_path() 误判修复** — 开发者布局检测逻辑在 skill 安装环境下误触发，导致 packages path 被错误注入。
2. **_brick_catalog_dir() 路径解析修复** — 候选路径顺序调整，优先读取 DORAMAGIC_BRICKS_DIR 环境变量，解决特定部署布局下查找失败的问题。
3. **发现阶段相关性过滤支持中文 repo** — 过滤器错误跳过中文描述的 GitHub repo（识别为空 description），修复后中文 repo 正确纳入发现结果。
4. **动态版本号 footer** — 不再硬编码版本号，改为运行时从 SKILL.md 读取，永远与实际安装版本一致。
