  Some skills have `disable-model-invocation` configured, which prevents
  them from being invoked via the Skill tool in Claude Code.
  Check skill configuration if you get this error.

  - "Check skill's SKILL.md for disable-model-invocation setting"
  - "Try different invocation method if available"
  - "Report to maintainer if blocking"

  - "Assume it's a user error"
  - "Keep trying the same invocation method"

## 真实场景
你尝试用 Skill tool 调用 brainstorming skill，但报错说 skill 配置了 disable-model-invocation。
这是 Claude Code 的限制，不是你的用法问题。

## 影响范围
- 谁会遇到：尝试使用某些特定 skills 的开发者
- 什么时候：调用特定 skills 时
- 多严重：部分功能无法使用，需要变通或等待修复
