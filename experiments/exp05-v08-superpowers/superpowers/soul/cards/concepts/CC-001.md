## Identity
The Skills System is Superpowers' core abstraction—a collection of composable, triggerable workflow modules that guide the AI agent through software engineering best practices at each development stage.

## Is / Is Not

| IS | IS NOT |
|----|--------|
| A workflow orchestrator that triggers based on context | A rigid pipeline that must be followed linearly |
| Composable (skills can be combined) | A monolithic workflow engine |
| Automatically triggered based on detected context | Manually invoked commands |
| Extensible (add new skills easily) | A fixed set of hardcoded rules |

## Key Attributes

| Attribute | Type | Purpose |
|-----------|------|---------|
| trigger_condition | string | When the skill activates (e.g., "before writing code") |
| workflow_steps | array | Ordered steps to execute |
| termination_condition | string | When the skill is considered complete |
| dependencies | array | Other skills that must complete first |

## Boundaries
- Starts at: Agent detects relevant context (e.g., user wants to implement a feature)
- Delegates to: Individual skill implementations (brainstorming, TDD, etc.)
- Does NOT handle: Direct code execution, file system operations
