rule: |
  Users cannot easily tell if Superpowers skills are being triggered.
  There's no clear visual feedback when a skill is activated.
do:
  - "Look for skill announcement: 'Using [skill] to [purpose]'"
  - "Check if TodoWrite todos are being created"
  - "Ask agent explicitly what skill it's using"

  - "Assume skills are working without verification"
  - "Ignore lack of feedback"

## 真实场景
你按照文档安装了 superpowers，但不确定它是否在工作。
和标准 plan mode 相比，你感觉不到任何区别。

## 影响范围
- 谁会遇到：所有用户，尤其是新用户
- 什么时候：使用过程中
- 多严重：用户不确定系统是否正常工作，影响信任度
