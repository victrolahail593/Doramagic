  Since v1.2.1, python-dotenv supports loading .env from FIFOs (named pipes).
  This enables dynamic configuration from external processes.

  FIFOs allow sending environment variables from other programs in real-time,
  useful for secrets injection from external secret managers.

## 真实场景
你想从 HashiCorp Vault 获取 secrets，通过 FIFO 将其传给 python-dotenv。
写一个后台进程不断向 .env_fifo 写入最新的 secrets，
你的 Python 应用启动时自动加载最新配置，无需重启。

## 影响范围
- 谁会遇到：需要动态 secrets 注入的开发者
- 什么时候：使用 Unix 系统时
- 多严重：低 - 高级功能，大部分用户用不到
