## Steps

1. **find_dotenv()** → `src/dotenv/main.py:L198-218`
   - Searches upward from cwd for .env file
   - Checks interactive mode to decide search strategy

2. **DotEnv() constructor** → `src/dotenv/main.py:L42-53`
   - Accepts dotenv_path or stream

3. **parse()** → `src/dotenv/main.py:L91-95`
   - Delegates to parser.parse_stream()
   - Returns iterator of Bindings

4. **set_as_environment_variables()** → `src/dotenv/main.py:L97-107`
   - Injects key-value pairs into os.environ
   - Respects override flag

## Flowchart

```
start
  ↓
find_dotenv() → DotEnv()
  ↓
parse() → Binding iterator
  ↓
interpolate? → YES → resolve ${VAR}
  ↓
set_as_environment_variables()
  ↓
end (returns bool)
```

## Failure Modes

| Failure | Cause | Handling |
|---------|-------|----------|
| File not found | No .env in search path | verbose=True warns, else silent |
| Parse error | Malformed line | error=True flag on Binding, logs warning |
| Permission denied | File mode issues | Raises OSError |
| Symlink attack | Malicious symlink | follow_symlinks=False by default |
