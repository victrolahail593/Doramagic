## Steps (set_key)

1. **Path resolution** → `src/dotenv/main.py:L143-152`
   - Resolve realpath, handle symlinks carefully
   - Default mode 0o600 if creating new

2. **Read existing content** → `src/dotenv/main.py:L162-172`
   - Read current .env file
   - Handle missing file gracefully

3. **Parse variables** → `src/dotenv/main.py:L178-187`
   - Use parse_variables() for ${VAR} expansion

4. **Rewrite file** → `src/dotenv/main.py:L189-215`
   - Find and replace existing key OR append new key
   - Preserve original file mode (since v1.2.2)
   - Handle quote modes (always/auto/never)

## Steps (unset_key)

1. **Parse file** → `src/dotenv/main.py:L217-230`
   - Load all key-value pairs

2. **Filter out target key** → `src/dotenv/main.py:L232-240`
   - Remove matching key from dict

3. **Rewrite file** → `src/dotenv/main.py:L242-255`
   - Write remaining keys back
   - Preserve comments and formatting

## Failure Modes

| Failure | Cause | Handling |
|---------|-------|----------|
| Key exists in symlink | Symlink target modification | follow_symlinks=False by default |
| File mode lost | Old behavior reset mode | v1.2.2 preserves mode |
| Concurrent modification | Race condition | No locking mechanism |
| Encoding issues | Non-UTF8 files | Specify encoding parameter |
