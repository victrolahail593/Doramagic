## Steps

1. **_is_interactive()** → `src/dotenv/main.py:L159-185`
   - Check if running in REPL/IPython/notebook
   - Checks sys.ps1, sys.ps2, __main__.__file__

2. **_is_debugger()** → `src/dotenv/main.py:L187-196`
   - Check if running under debugger

3. **_walk_to_root()** → `src/dotenv/main.py:L133-141`
   - Walk upward from starting directory

4. **find_dotenv()** → `src/dotenv/main.py:L198-218`
   - Search for .env file in parent directories
   - Uses usecwd=True to include current directory

## Flowchart

```
start
  ↓
_is_interactive()?
  ↓
_is_debugger()?
  ↓
_walk_to_root() from starting point
  ↓
check each dir for .env
  ↓
return first match or ""
```

## Failure Modes

| Failure | Cause | Handling |
|---------|-------|----------|
| No file found | No .env in tree | Returns empty string |
| cwd doesn't exist | Deleted working directory | Returns None |
| Permission denied | Cannot read directory | Continues to parent |
