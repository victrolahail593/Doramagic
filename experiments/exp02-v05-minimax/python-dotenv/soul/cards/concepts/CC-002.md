## Identity
The parser is the engine that understands .env file syntax. It converts raw text into structured key-value bindings with proper escape sequence handling.

## Is / Is Not

| IS | IS NOT |
|----|--------|
| A tokenizing reader for .env syntax | A full-fledged configuration file reader |
| Handles quotes, escapes, comments | Handles validation of values |
| Returns iterator of Bindings | Modifies environment directly |

## Key Components

| Component | Purpose |
|-----------|---------|
| Reader class | Character-by-character text consumption |
| Binding NamedTuple | Holds key, value, original string, error flag |
| parse_stream() | Main entry point returning Binding iterator |
| Escape handling | Supports \\, \', \" in quotes |

## Boundaries
- Input: Raw text stream
- Output: Iterator of (key, value, original, error) tuples
- Does NOT handle: File I/O, environment injection
