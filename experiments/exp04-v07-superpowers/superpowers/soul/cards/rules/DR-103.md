rule: |
  Superpowers skills add prompt overhead that increases token usage.
  Some users report burning too many tokens on skill content, especially
  for long conversations. Consider: 1) Use @ links carefully,
  2) Skills evolve - don't 'remember' old versions, 3) Lean context options.
do:
  - "Load only relevant skills, not all at once"
  - "Use skill tool to load specific skill when needed"
  - "Check for skill updates periodically"

  - "Load all skills at session start"
  - "Force-load skills with @ syntax (burns 200k+ context)"
  - "Ignore token efficiency for long projects"

## 真实场景
用户和 superpowers 进行了长对话。一开始就加载了所有 skills。
100 条消息后，token 消耗远超预期。每次 agent 回复都要处理大量 skill 内容，
速度变慢，成本变高。用户想知道是否可以更高效。

## 影响范围
- 谁会遇到：长期使用 superpowers 的用户
- 什么时候：长对话或多轮交互时
- 多严重：MEDIUM — 增加成本，降低响应速度
