## Identity
The Skills System is the foundational mechanism that forces AI agents to follow structured workflows by automatically invoking relevant skills BEFORE any action, even clarifying questions.

## Is / Is Not

| IS | IS NOT |
|----|--------|
| A mandatory workflow enforcement layer | Optional suggestions or tips |
| A skill discovery and invocation system | A skill creation framework |
| Triggered by context analysis (1% rule) | A manual command dispatcher only |
| Platform-agnostic (Claude Code, Codex, Cursor, OpenCode) | Vendor-specific implementation |

## Key Attributes

| Attribute | Purpose |
|-----------|---------|
| Skill tool | Platform-native mechanism to load skill content |
| 1% rule | If there's any chance a skill applies, invoke it |
| Skill priority | Process skills first, then implementation skills |
| Red flag detection | Identifies rationalization to skip skills |

## Boundaries
- Starts at: User message received
- Enforces: Skill check BEFORE any response or action
- Does NOT: Write code, create files, or take implementation actions without skill invocation first
