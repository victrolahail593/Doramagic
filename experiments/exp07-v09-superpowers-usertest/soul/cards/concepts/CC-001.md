## Identity
Skills 是 superpowers 的核心机制——它们是独立的 Markdown 文件，包含触发条件（description）、执行指令和最佳实践。当代理识别到匹配上下文时自动加载并执行。

## Is / Is Not

| IS | IS NOT |
|----|--------|
| 上下文感知的自动触发器 | 手动调用的一次性脚本 |
| 可组合的工作流单元 | 硬编码的固定流程 |
| 跨平台兼容（Claude Code/Cursor/Codex/OpenCode） | 某个 IDE 的专有插件 |

## Key Attributes

| Attribute | Type | Purpose |
|-----------|------|---------|
| name | string | 技能唯一标识符 |
| description | string | 触发条件描述，支持自然语言匹配 |
| model | string | 可选的模型覆盖 |
| tags | string[] | 分类标签 |

## Boundaries
- 触发：基于 description 关键词匹配
- 执行：加载后完全按照指令执行
- 不处理：持久化状态（依赖外部 TodoWrite）

## Evidence
- Skills 目录结构: skills/brainstorming/SKILL.md, skills/test-driven-development/SKILL.md 等
- 触发规则: skills/using-superpowers/SKILL.md:L1-20
- 跨平台支持: .claude-plugin/plugin.json, .cursor-plugin/plugin.json, .codex/INSTALL.md, .opencode/INSTALL.md
