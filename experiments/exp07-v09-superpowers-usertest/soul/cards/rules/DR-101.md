## 真实场景
你按照 README 安装了 superpowers，然后说"用 brainstorming skill 帮我设计这个功能"。Claude Code 回复说正在使用 skill，但 skill 内容是空的——它只是假装在用。实际上是 skill 根本没有正确加载。

## 影响范围
- 谁会遇到：新手用户、安装过程有问题的用户
- 什么时候：第一次安装后
- 多严重：Skill 完全失效，整个 superpowers 工作流无法使用
