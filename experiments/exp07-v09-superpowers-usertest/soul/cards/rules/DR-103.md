## 真实场景
你把 superpowers clone 到了 ~/.config/opencode/，但 Claude Code 是在 ~/.claude/plugins/ 里找 skills。你运行任何需要 skill 的命令都说找不到。你反复检查了 README，确认每一步都做了，但还是不行——因为路径完全错了。

## 影响范围
- 谁会遇到：跨平台用户、不熟悉各平台约定的新手
- 什么时候：安装过程中
- 多严重：Skills 完全不可用
