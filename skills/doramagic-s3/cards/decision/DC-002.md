# Decision: Data Export Format

## Context
用户需要备份或迁移他们收集到的 WiFi 密码。

## Rule
- **Do**: 提供结构化格式（如 JSON）导出。
- **Don't**: 将明文密码未经警告随意导出到公共目录。

## Rationale
- **WHY**: 便于跨设备恢复和被第三方密码管理器（如 Bitwarden/1Password）解析。社区中有提出加密导出的需求（Issue #15），说明纯明文导出存在安全担忧。
