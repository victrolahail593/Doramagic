# Decision: Prefer Shizuku over Root

## Context
在 Android 上读取系统级别的 WiFi 密码需要高权限。传统方式是 Root。

## Rule (Do/Don't)
- **Do**: 提供并优先推荐使用 Shizuku 获取权限。
- **Don't**: 强制用户必须 Root 才能使用。

## Rationale
- **WHY**: Root 设备正在变得越来越困难且会导致许多银行/游戏应用无法运行。Shizuku 利用无线 ADB，提供了一种无需修改系统分区的安全高权限获取途径，极大降低了用户的使用门槛（共识）。
