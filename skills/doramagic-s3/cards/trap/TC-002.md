# Trap: SSID vs Network Name Mismatch

## Description
在生成 WiFi 连接二维码时，如果错误地将网络的显示名称（Network Name）而不是真实的广播标识符（SSID）放入二维码的数据结构中，会导致手机扫码后无法连接。

## Source
Issue #1 (network-manager-qrcode): "Create barcode puts wrong SSID when SSID!=Network name"

## Severity
Critical - 核心功能失效
